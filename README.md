# Nublo Weather App

A Flutter weather application with map integration.

## How to Run

1. Install dependencies:
```bash
flutter pub get
```

2. Generate code:
```bash
dart run build_runner build --delete-conflicting-outputs
```

3. Run the app:
```bash
flutter run
```
